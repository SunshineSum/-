###获取class列表

       add( String )
			添加指定的类值。如果这些类已经存在于元素的属性中，那么它们将被忽略。
	   remove( String )
			删除指定的类值。
	   toggle ( String )
			切换 class value; 即如果类存在，则删除它并返回false，
			如果不存在，则添加它并返回true
			如果存在就会自动删除，如果没有存在就自动给我们加上来
